Overview

This page describes our testing process.

Usability

We performed usability testing on our UAT instance.

Unit

All architecturally significant components have associated unit tests defined in the test/unit directory. We utilize the Spock framework, which is part of Grails. Unit tests are executed for each Git check-in, in the label-unit Jenkins build.

Integration

All architecturally significant components have associated integration tests defined in the test/integration directory. We utilize the Spock framework, which is part of Grails. Integration tests are executed for each Git check-in, in the label-integration Jenkins build, as a child of the label-unit build.

Functional

All significant UI functionality is test through browser automation tests using the Geb framework, for each supported browser. Functional tests are in the test/functional directory. Functional tests are executed for each Git check-in, in the label-functional Jenkins build, as a child of the label-integration build.
