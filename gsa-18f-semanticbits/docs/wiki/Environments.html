Continuous Monitoring

We utilize a number of AWS services to provide continuous monitoring on Label.  In other projects, we utilize additional services, such as ThreatStack.

CloudWatch

AWS CloudWatch provides us real-time visibility into cloud assets, providing us up-to-the-minute information about our overall physical architecture.


CloudTrail

AWS CloudTrail provides us visibility into access of cloud resources, including real-time notifications of access events.  The AWS API call history produced by CloudTrail enables security analysis, resource change tracking, and compliance auditing.


Config

AWS Config provides us resource inventory, configuration history, and configuration change notifications to enable security and governance.  These capabilities enable compliance auditing, security analysis, resource change tracking, and troubleshooting.
